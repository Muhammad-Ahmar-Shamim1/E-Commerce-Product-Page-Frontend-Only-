E-Commerce Product Page – Frontend Web Project

This project is a responsive e-commerce product page developed using only HTML5, CSS3, and JavaScript. It includes a fully working shopping cart system with quantity controls, automatic price calculation, and browser-based data storage using localStorage.

The entire application works without any backend or external libraries.

Project Objective

The main objective of this project is to practice frontend development by building a real-world shopping interface using pure JavaScript and modern CSS layout techniques.

Technologies Used

HTML5 – For page structure and semantic elements

CSS3 – For layout, animations, and responsiveness

JavaScript – For cart functionality and dynamic rendering

Browser localStorage – To save cart data locally

Features
Product Listing

Displays four products dynamically from JavaScript.

Each product shows image, name, description, and price.

Responsive grid layout adapts for mobile, tablet, and desktop screens.

Shopping Cart

Add items to the cart with one click.

Duplicate products increase quantity instead of repeating.

Quantity can be increased or decreased using buttons.

Items can be removed from the cart.

Total price updates automatically.

Empty cart message appears when no items exist.

Data Persistence

Cart data is saved in the browser using localStorage.

Cart items remain even after page reload.

User Experience

Toast notification when items are added.

Smooth fade-in animations.

Hover effects on product cards.

ESC key closes the cart sidebar.

Fully responsive on all devices.
